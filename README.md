# Fixed-Wing UAV: AI Detection Competition

**TEKNOFEST 2026 — Proposal Stage (not built)**

## Overview

This project proposed a remote-controlled fixed-wing UAV for a TEKNOFEST competition category that scores aircraft on detecting opposing drones via computer vision, while minimizing the platform's own visual detectability to opposing teams' AI systems. The competition is non-weaponized; scoring is based purely on detection performance. The mission sequence begins with the aircraft reading a QR code to trigger competition start.

## Status

This project's technical proposal did not advance past TEKNOFEST's review stage, so it did not receive competition funding and was not physically built. The work in this repository reflects the design and engineering work completed up to that point.

## What's in this repository

- **Airframe design** — lightweight fixed-wing structural concept balancing weight, speed, and onboard payload (camera/electronics) capacity
- **Detection system planning** — computer-vision approach for identifying competing aircraft via onboard camera feed
- **Signature design notes** — airframe shape and surface trade-offs considered for reducing visual detectability
- **Mission sequence design** — QR-code-triggered start sequence integrated into the competition mission plan

## What we learned

The detection-vs-evasion trade-off required balancing structural weight, speed, and payload capacity in ways that needed further iteration before the design was competition-ready. The proposal did not advance past technical review; this repository documents the design work completed through that stage.
